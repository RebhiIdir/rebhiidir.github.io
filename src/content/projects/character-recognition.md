## Objective

Compare a classical machine learning approach (K-Nearest Neighbors) against a deep learning approach (CNN) on the same character recognition task, to build intuition on when each paradigm is worth its cost.

## Technical scope

- KNN classifier with tuned distance metric and neighborhood size.
- Convolutional neural network trained on the same character dataset.
- Side-by-side accuracy and error analysis between the two approaches.
